# tabbar_demo
小程序自定义tabbar demo

因小程序提供的tabbar在功能，样式上都有限制，故实现自定义tabbar
实现原理是在所有一级页面引入tabbar，从而实现页面tabbar效果
但每次切换页面都会重新加载tabbar组件，所以自定义tabbar存在切换时页面闪动的问题
